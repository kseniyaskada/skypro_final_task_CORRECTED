Git

  • git init — создает новый локальный репозиторий.
  • git clone [ссылка] — клонирует удаленный репозиторий в локальную директорию.
  • git remote add origin [ссылка] — связывает локальный и удаленный репозитории.
  • git status — показывает статус файлов (измененные, новые, удаленные).
  • git commit -m "сообщение" — фиксирует изменения с комментарием.
  • git pull — получает последние изменения из удаленного репозитория.
  • git push — отправляет локальные изменения в удаленный репозиторий.
  • git switch --create [имя] — создает новую ветку и переключается на нее
  • git switch [имя] — переключается на существующую ветку
  • git branch — показывает список всех веток
